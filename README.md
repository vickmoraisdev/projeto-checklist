# ✅ Checklist

Uma aplicação de Checklist desenvolvida com HTML, CSS e JavaScript, permitindo adicionar, concluir e remover tarefas de forma simples e intuitiva.

## 🚀 Demonstração

Acesse o projeto online:

🔗 https://vickmoraisdev.github.io/projeto-checklist/

## 📸 Preview

![Preview do projeto](https://github.com/user-attachments/assets/ca9cf766-6bb1-4dc6-8e8f-c4b991f16a42)

## ✨ Funcionalidades

* Adicionar novas tarefas
* Marcar tarefas como concluídas
* Remover tarefas da lista
* Feedback visual para ações do usuário
* Interface responsiva

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript

## 📚 Aprendizados

Durante o desenvolvimento deste projeto, foi possível praticar conceitos importantes do desenvolvimento Front-End, como:

* Manipulação do DOM
* Eventos JavaScript
* Criação dinâmica de elementos
* Estruturação e organização de código
* Responsividade

## 🎯 Objetivo do Projeto

Este projeto foi desenvolvido com o objetivo de fortalecer conhecimentos em JavaScript, aplicando conceitos fundamentais em uma aplicação prática e interativa.
